# CareerOS

CareerOS Landing Page

A modern, responsive landing page built with HTML, TailwindCSS, and JavaScript. CareerOS helps students manage career opportunities efficiently, ensuring no opportunity is missed.

Features

Fully responsive design for desktop and mobile. Smooth scroll and reveal animations. Interactive cursor glow effect. Glass-card UI components with hover animations. Step-by-step workflow display. Waitlist form with dynamic submission feedback. Custom scrollbar styling. Modern color palette and gradient text highlights. Semantic HTML with accessible components.

Technologies Used

HTML5 – Page structure and semantic markup. TailwindCSS – Rapid, utility-first styling. Google Fonts (Inter) – Clean and professional typography. Vanilla JavaScript – Interactivity for: Cursor glow effect Scroll-based reveal animations Navbar scroll effect Waitlist form submission feedback
